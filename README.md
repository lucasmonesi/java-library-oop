# java-library-oop

Projeto desenvolvido para reforçar fundamentos de Java antes de iniciar estudos com Spring Boot.

A ideia foi praticar orientação a objetos e regras de negócio sem depender de frameworks.

## O que foi praticado

- Uso de HashMap para armazenamento em memória
- Encapsulamento
- Controle de estado da entidade (livro emprestado ou disponível)
- Validação de regras de negócio
- Separação básica de responsabilidades

## Funcionalidades

- Cadastrar livro
- Buscar por ID
- Emprestar e devolver
- Listar livros disponíveis
- Remover livro
